
In addition to the packages listed in requirements.txt, this project also uses Python 3.10.11.
Newer versions may cause errors.

To input your account information, delays, proxy, and name list, edit the configuration files in the Config folder.

If you want to use a proxy, it must use this format:
      server:port:username:password
For example: 
      "26012PVwg3-resi_region-US_Virginia_Ashburn:IJLR8tGYD3k@ca.proxy-jet.io:1010"
If you don't want to use a proxy, just leave that string empty.
